东方绮符剧常见问题及联机教程

1.   常见问题

  1. 有时如果无法打开游戏，请检查任务管理器，找到 QSanguosha.exe 这个进程，将其关闭再启动即可。
  2. 本游戏需要VC2015支持，如出现无法打开游戏的提示( 多数情况为找不到msvcp140.dll )，请自行百度或在以下链接下载：

VC2015 运行库在本游戏下载地址的百度盘文件夹里同时附带了

贴士：下载x86的那个(哪怕你是64位的系统，x64的运行库或许也没有用，下x86的即可)。

另外有些人可能会出现这种情况：提示安装不成功，但是就算如此你也已经可以开本游戏了。

  1. 本游戏对XP系统需要额外兼容，如果 XP系统出现兼容问题 ，请私聊官方群群主或管理员Slob，会帮你们解决。
  2. 加入游戏时，点击"连接"按钮， 只可以单击 ，然后等待连接，切不可双击或多次点击，否则会造成很多链接问题。
  3. 加入游戏时，请务必勾选上" 断线重连 "。
  4. 使用游侠对战平台如果无法启动服务器，请使用 兼容模式 打开平台。
  5. 游戏时请善用以下快捷键：" D "键可以在其他角色头像上显示/隐藏你和其他角色的距离，" Z "键可以显示/隐藏技能按钮(方便欣赏头像)。
  6. 在查看武将列表，进行搜索时，需要使用"*"和"?"代表通配符号进行搜索，比如输入"*丽*"就可以搜索到所有中间含有"丽"这个字的角色啦。
  7. 发现任何bug问题，请立即截图， 截图必须包括右上角的流程记录和全场所有角色及其装备和状态 ，以帮助程序员重现这个bug，这很重要！
  8. 如果有任何意见建议或反馈信息，请务必私聊"小巫女莎莎"或其他管理员，将你的反馈传达给制作人员，少了你们的反馈这个游戏永远都会停滞不前。（尤其不要光在QQ群里说，我们不可能时刻看着群里面，请私聊反馈。）
  9. 有意见建议也可以去"东方绮符剧"吧发帖。同时关注该贴吧，因为会不定时在那里发布各种新版本信息或举办一些活动。
  10. 请仔细阅读其他文档里的内容，否则后果自负。
  11. 游戏建议使用永久IP地址。具体见群公告。

2.   关于联机( 当你加入 其他 玩家 的主机时 )

  1. 点击主界面上的“ 加入游戏 ”选项
  2. 设置好自己的“ ID名 ”和“ 头像 ”
  3. < 重要 >请一定要勾上“ 断线重连 ”
  4. 在“ 主机 ”一栏，输入主机IP地址， 在群公告或游戏自带文本里有提供已经建好的永久IP地址，可以选择使用那个联机 ，然后点击“ 连接 ”
  5. < 极其重要 >在点击“ 连接 ”时请一定仅仅 单击 ， 请一定不要双击该按钮 ，单击之后等待连接即可，否则后果自负。

3.   关于联机( 当你 自己 做 主机时 )

  1. 确保自己有外网IP（如何分辨自己是不是外网IP可以百度）(那些192.168开头的肯定是内网IP乖乖去用群内永久主机地址)，没有外网IP的话，可以百度“ 端口映射 ”尝试自己映射一个。还可以去选择用“ 游侠对战平台 ”。
  2. 点击主界面上的“ 启动服务器 ”选项
  3. 在“ 基本 ”选项卡中，可以选择操作时间，如果勾上“没有限制”则没有操作时间限制。当前测试版本所有非常规身份局的模式均不可以选择，请期待后续版本。
  4. 在“ 游戏包选择 ”选项卡中可以选择自己希望加入游戏的人物。“ 神祝 ”包为所有神将，含有“ 异世界 ”的包为非东方的角色，可以视喜好勾选。
  5. 在“ 高级 ”选项卡中，最大洗牌次数可以调大一些（达到最大洗牌次数视为平局游戏结束）。选将上限当前版本已锁定。
  6. 在“ 高级 ”选项卡中，请 千万不要 选择“双将”“暗将”模式，出现bug不负责（当前版本已锁定不可选择）。在“其他”选项卡中没有什么需要调整的。
  7. 点击“ 启动服务器 ”则建立主机，然后把你的公网IP发给一起玩的小伙伴。
  8. 重新开始游戏，你自己在 第>3<大项 中按照操作连接你主机中的“ 回环地址 ”即可，其他人则需连接你的外网IP。
  9. 开始游戏，等待其他玩家加入

4.   关于游侠对战平台

当群内永久主机地址没有希望游戏的模式时 ，因为很多很多小伙伴，都没有外网IP，所以可以选择进入 “ 游侠对战平台 ”进行联机游戏。 程序在主群群文件内可以找到 。

  1. 先注册登录
  2. 进入平台，进入“ 策略战棋 ”>“ 太阳神三国杀 ”房间
  3. 所有人选择同一个专房，然后点 “ 开始游戏 ”，第一次开的童鞋将绮符剧的路径贴上去。
  4. 然后其中一个童鞋如 第>4<大项 中按照操作建立主机
  5. 其他人在绮符剧主界面点击“ 加入游戏 ”，然后选择头像和昵称，并勾上“ 断线重连 ”
  6. 点击中间的“ 探索局域网 ”点刷新，找到建主的IP，若搜索不到就直接输入主机IP（将光标悬停至该角色在游侠内的用户名上方即可查看虚拟IP地址）
  7. 开始游戏，等待其他玩家加入
  8. 关于使用游侠对战平台无法启动服务器的解决办法： 请使用“兼容模式”启动游侠对战平台或绮符剧本体 ；如果使用兼容模式依然无法启动，请检查 任务管理器 ，找到 QSanguosha.exe 这个进程， 将其关闭再启动 即可。

5.   关于单机

在“ 启动服务器 ”中，设置完毕后，最下有个“ 单机启动 ”的按钮，点击即可。（ 重要声明 ：该软件对AI的编写还存在部分问题，所以可能会出现bug和电脑错误操作，如不使用技能或乱打。所以除非用于熟悉技能，其他情况请建议和真人进行游戏）

6.   关于字体

推荐使用“ 文泉驿微米黑 ”或“ 华康娃娃体20M完整版 ”，前者制作非常精美同时开源，作为系统字体非常帅；后者胜在字体可爱并且字库极其庞大，任何生僻字都不会出现显示不出来的情况。

如果觉得字体锯齿多不美观，建议下载 Mactype 进行渲染，非常有用的字体渲染软件。这边附上两款字体的下载和莎莎的Mactype的配置文件。 链接地址 ：http://pan.baidu.com/s/1lMxxc
